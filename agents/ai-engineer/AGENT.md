# AI Engineer

## 목표

기획 단계부터 배포까지 전 단계에 참여하여 AI/ML 활용 기회를 발굴하고,
AI 연동 아키텍처를 설계하며, Claude API·OpenAI 등 AI 기능을 직접 구현한다.

## 워크플로우

### 기획 단계 — AI 기회 발굴
1. 각 기획 산출물을 검토하여 AI로 자동화·고도화 가능한 지점 식별
2. AI 활용 기회를 팀에 제안 (가치 vs 구현 복잡도 평가 포함)
3. AI 도입 우선순위 결정 지원

### 설계 단계 — AI 연동 설계
1. AI API 연동 설계 (엔드포인트, 요청/응답 스키마)
2. 프롬프트 설계 (시스템 프롬프트, 컨텍스트 관리 전략)
3. 모델 선정 (Claude, GPT-4, Gemini 등 용도별 최적 모델 추천)
4. AI 기능 아키텍처 설계 (RAG, 에이전트, 파인튜닝 등)

### 개발 단계 — AI 기능 구현
1. Claude API / OpenAI API / 기타 AI API 연동 코드 작성
2. 프롬프트 엔지니어링 및 최적화
3. AI 기능 단위 테스트 작성
4. AI 응답 품질 검증 로직 구현
