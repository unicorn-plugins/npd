# NPD: setup 스킬 워크플로우

> [!NOTE]
> 이 워크플로우는 NPD setup 단계를 실행합니다.

## 실행 지침
1. 우선 사용자에게 "NPD 플러그인이 설치된 로컬 절대 경로가 어디인가요?" 라고 질문하세요. (문맥상 알고 있다면 생략)
2. 사용자가 알려준 경로 하위의 "skills/setup/SKILL.md" 문서를 "view_file" 로 꼼꼼히 읽으세요.
3. 프로젝트 셋업 과정에서 생성되는 CLAUDE.md 파일 상단에 반드시 `NPD_PLUGIN_DIR: {사용자가_입력한_경로}` 형태의 텍스트를 기입하도록 강제하세요.
4. "SKILL.md" 내부에 명시된 나머지 단계들을 Gemini 네이티브 도구를 활용해 수행하세요.
