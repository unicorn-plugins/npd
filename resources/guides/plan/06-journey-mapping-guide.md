# 고객 여정 맵 작성 가이드

## 목적

고객의 여정을 시각화하고 분석하는 User Journey Map을 작성함.

## 여정 맵 구성 요소

### 1. 페르소나
대표 사용자를 정의함:
- 이름, 나이, 직업
- 배경 및 상황
- 목표 및 동기
- 기술 숙련도

### 2. 고객경험 단계 (X축)
customer-journey-stages에서 정의한 고객경험 단계를 X축으로 사용함.

### 3. 단계별 분석 항목

각 단계를 다음 요소로 분석함:
- **고객 행동**: 무엇을 하는가?
- **생각**: 무엇을 생각하는가?
- **감정**: 어떻게 느끼는가? (감정 곡선)
- **터치포인트**: 어디에서 서비스와 상호작용하는가?
- **Pain Points**: 어려움과 장애물
- **Gain Points**: 긍정적 경험과 만족 요소

### 4. 핵심 인사이트
- 가장 큰 Pain Points 상위 3개
- 개선 기회 영역
- 감정의 고점과 저점
- 이탈 위험 지점

### 5. 기회 영역
- 높은 우선순위 개선 영역
- 혁신이 필요한 터치포인트
- 차별화 포인트

## 감정 곡선 표현

```
 높음 ┤     ╭─╮
      │    ╱   ╲    ╭─╮
      │   ╱     ╲  ╱   ╲
 중간 ┤  ╱       ╲╱     ╰─╮
      │ ╱                  ╲
 낮음 ┤╯                    ╰
      └─────────────────────────>
       단계1  단계2  단계3  단계4  단계5
```

## 시각화 가이드라인

- 감정 상태 표현: 긍정 / 중립 / 부정
- 감정 레벨: 5점 만점
- 명확한 단계 경계 사용
- 중요한 순간 강조
- 시간 경과에 따른 감정 궤적 표시

## 주의사항

- 고객경험 단계를 X축으로 사용
- 실제 고객 데이터 기반 작성 (추측 X)
- 감정 곡선은 정량적 데이터 기반
- Pain Point는 구체적이고 측정 가능하게
- 터치포인트는 실제 접점 위주로
- 개선 기회는 실행 가능하게
- SVG 파일로 시각화 포함
