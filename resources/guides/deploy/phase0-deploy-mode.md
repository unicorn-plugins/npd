# Phase 0 진행 모드 선택 가이드

> 본 가이드는 `skills/deploy/SKILL.md > Phase 0`에서 호출됨.
> 진행 모드(단계별 승인/자동 진행)와 자동 진행 모드 시 VM Host 사전 수집 절차를 정의.

## 입력

- `~/.ssh/config` (자동 진행 모드 시 VM Host 후보 파싱)

## 공통 원칙

- **자동 진행 보장**: Phase 1~6 실행 중 사용자 입력이 필요한 항목을 본 Phase에서 사전에 모두 수집
- **저장 위치**: 진행 모드 결정 결과는 `{PROJECT_DIR}/AGENTS.md > ## 워크플로우 진행상황 > ### deploy > 진행 모드`에 기록

---

- **단계별 승인** 선택 시 → 각 스텝 완료 후 아래 ASK_USER로 승인 요청을 표시하고 응답에 따라 분기:
  - **승인** → 다음 스텝 진행
  - **재작업 요청** → 사용자 피드백을 받아 현재 스텝 재수행
  - **중단** → 현재까지 산출물 보존 후 스킬 종료
- **자동 진행** 선택 시 → Phase 0(진행 모드·VM Host 사전 수집)에서 필요한 정보를 모두 수집한 뒤, 워크플로우 실행 중(Phase 1 ~ Phase 6)에는 `<!--ASK_USER-->`를 호출하지 않고 아래 규칙으로 자동 처리한다:
  - **단계 간 승인**: 생략 (연속 실행)
  - **자동 감지 가능한 정보** (Cloud CLI 로그인 여부, SSH 접속 테스트, Web Server 설치 여부, SSL 도메인 등): 명령어 실행 결과로 자동 판단하고, 실패 시에만 사용자에게 질문
  - **VM Host 선택**: `~/.ssh/config`에 후보가 1개면 자동 선택, 2개 이상이면 Step 2에서 미리 수집
  - **기타 선택지**: 기본값 또는 자동 감지 결과를 사용

## Step 2. VM Host 사전 수집 (자동 진행 모드)

> **조건**: 자동 진행 모드이고, `~/.ssh/config`에 Host 후보가 2개 이상일 때만 수행

`~/.ssh/config`를 파싱하여 후보가 2개 이상이면 여기서 미리 선택받는다:
